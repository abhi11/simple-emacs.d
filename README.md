##Simple Emacs configuration##
####Emacs 24

###Clone it to your system and start using###

###Steps###
* Clone repo to your home directory.
* Re-name the simple-emacs.d to .emacs.d/
* Start your emacs.

###List of packages configured.###
* auto-complete
* YASnippet
* Markdown-mode
* Magit
* cider
* zenburn theme
